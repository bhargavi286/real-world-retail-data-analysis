# Real-World Retail Data Analysis Project

## Overview

This project focuses on analyzing a real-world retail sales dataset to uncover business insights and trends. The analysis includes sales performance, profit analysis, regional comparisons, category-wise analysis, and correlation studies using data visualization techniques.

## Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Project Objectives

* Analyze retail sales data
* Identify key business trends
* Evaluate category and regional performance
* Visualize important metrics
* Generate actionable insights

## Analysis Performed

### Dataset Exploration

* Dataset shape and structure analysis
* Statistical summary generation
* Feature inspection

### Business Analysis

* Total Sales Analysis
* Total Profit Analysis
* Category-wise Sales Analysis
* Category-wise Profit Analysis
* Region-wise Sales Analysis

### Correlation Analysis

* Correlation Matrix
* Heatmap Visualization

## Visualizations

1. Sales by Category
2. Profit by Category
3. Sales by Region
4. Correlation Heatmap

## Key Insights

* Different product categories contribute differently to overall sales and profit.
* Regional performance varies significantly across the business.
* Correlation analysis helps identify relationships among sales, quantity, discount, and profit.
* Data visualization supports better business decision-making.
